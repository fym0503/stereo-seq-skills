# Validation Checks

- Domain maps must use equal-aspect spatial coordinates.
- Check each domain for marker support, not only cluster identity.
- Compare domain boundaries to histology, ssDNA/H&E, or known anatomy when available.
- Verify that domain labels are stable across nearby resolutions or sections when claims require generality.
- Keep de novo clusters separate from curated biological domain labels.
- For cross-section transfer, report the reference section and mapping assumptions.

Warning signs:
- domains scattered randomly across tissue;
- domain boundaries follow sequencing depth or mitochondrial percentage;
- a claimed anatomical layer lacks layer markers;
- condition comparisons use different tissue masks or domain definitions.
