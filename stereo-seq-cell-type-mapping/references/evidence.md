# Pilot Evidence

The strongest evidence for this skill comes from repeated deconvolution and label-transfer patterns in the 10-paper pilot.

- P01: RCTD v1.2.0, `doublet_mode="multi"`, bin50 cell composition; scatterpie visualization of composition.
- P03: public GitHub `single-cell-BGI/MBA` includes SPOTlight workflow for binned Stereo-seq data.
- P04: SPOTlight v0.1.6, bin50 pseudo-spots, top-four probabilities pruned and renormalized, primary cell type assigned for visualization.
- P05: spacexr/RCTD full mode on bin100 Stereo-seq slices with scRNA-seq fibroblast reference.
- P07: cell2location for trophoblast and decidua mapping; Tangram for representative-section subregion mapping.
- P09: Zenodo code includes SingleR and annotation scripts; use as code-level evidence pending workflow confirmation.
- P10: cell2location v0.1.3 integrating snRNA-seq and Stereo-seq for cell distribution.

Use `pilot/analysis/skill_evidence_matrix.tsv` in the source repo for a row-level evidence table.
