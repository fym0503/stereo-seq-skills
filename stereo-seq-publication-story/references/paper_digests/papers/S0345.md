STAGUE processes the input ST data through three views: the learner view, which combines expression features with spatial information to infer a cell adjacency matrix A capturing cell dependencies; the positive view, derived from the raw kNN cell adjacency matrix Γ; and the negative view, formed by shuffling cell features and using an identity adjacency matrix.
