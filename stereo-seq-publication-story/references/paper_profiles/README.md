# Stereo-seq Paper Profiles

This directory contains unified one-paper-one-Markdown profiles generated from the local Stereo-seq literature corpus.

Each `papers/S*.md` file combines objective corpus evidence and story/reuse cues for the same article. The skill should search this index first, then load only the few profiles relevant to the user's dataset or question.

- `papers/S*.md`: one unified profile per paper.
- `../paper_profile_all_index.tsv`: tabular index for all profiles.
- Source tables: `../paper_skill_matrix.tsv` and `../paper_figures.tsv`.
