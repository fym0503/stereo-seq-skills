# Validation Checks

- State the comparison groups or spatial graph clearly.
- Separate SVG/module discovery from DEG and enrichment.
- Record multiple-testing thresholds and module cutoffs.
- Check whether spatial programs correlate with UMI depth, tissue mask, or mitochondrial percentage.
- Map module or pathway scores back to coordinates before making spatial claims.
- Use domain/cell-type labels only if they were validated upstream.
- For regulons, confirm species-appropriate motif databases and TF gene symbols.
- For factor models, report rank choice and whether factors are stable.
